Responsive vs Adaptive

Ce inseamna machetare receptivă (Responsive)

Design responsive înseamnă un singur site web cu funcții de aspect pentru desktop și vizitatorii de pe mobil, deoarece elementele paginii web se adaptează „în mod receptiv” ecranului.

Trucul pentru a construi site-uri receptive (realizate de obicei în HTML și CSS sau HTML5 și CSS3) este utilizarea interogărilor pentru a viza punctele de întrerupere - zone în care un site se adaptează la diferite lățimi ale dispozitivului. Aceste puncte de întrerupere scalează imaginile, împachetează textul și ajustează aspectul pentru a se potrivi oricărui ecran.

Ce este designul adaptiv?

Pentru site-urile adaptive, designerii creează numeroase aspecte fixe pentru a se potrivi cu dispozitivul pe care este vizualizat un site.

De regula se folosesc "breakpoints", dimensiuni fixe in px, atunci cand design-ul sufera modificari:

  • Pentru mobile: 320px → 480px
  • iPads si Tablete 481px→ 768px
  • Laptop-uri si monitoare mici: 769px → 1024px
  • Ecrane mari si "Desktops": 1025px → 1200px
  • TV si Monitoare mari: 1201px and more
